## 개요

- 시작일: 2026-05-04
- 목표: 최소 14일 연속 사용 + 코치 제안에 반응한 횟수 5건 이상
- 측정 대상

  - 코치 제안 수락률
  - 수락 후 7일 완료율 변화 (stats 페이지의 acceptImpact 카드 스크린샷)
- 최종 산출물
  - 이 일지 전체
  - 주차별 stats 페이지 스크린샷
  - 발견한 이슈 → 수정 PR 연결 (최소 5건 목표)

## 작성 가이드

매일 한 번 기록. 30초~1분 안에 끝낼 만큼 짧게. 못 쓴 날은 비워두지 말고 '쉼' 한 줄이라도 남길 것.
빠뜨리면 안 되는 한 가지: **코치 제안을 받았다면 어떻게 반응했는지 + 이유.**

## 일일 기록 템플릿

```
### n일차 (YYYY-MM-DD)
- 체크: x/y (해당일 완료/전체 습관)
- 코치: 받음 → 수락|거부|무시 / 안 받음
  - 제안: <한 줄>
  - 이유: <왜 그렇게 반응했는지 한 줄>
- 메모: <발견한 버그·아이디어·기분·기타. 없으면 생략>
```
